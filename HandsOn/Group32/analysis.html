Datasets Analysis

Adress Directory

  • CSV Name: CALGIS.ADDRESS.csv
  • Description: It contains all adreesses with latitude, longitude, number of houses, street name,...

Fire Stations Directory

  • CSV Name: CALGIS.ADM_FIRE_STATION_OFFICE.csv
  • Description: It contains all fire stations street locations.

Census of Calgis

  • CSV Name: CALGIS.CENSUS_COMMUNITY_DIST_2015.csv
  • Description: It contains all information about the census of the city.

Amenity by places

  • CSV Name: CALGIS.CITY_AMENITY.csv
  • Description: It contains all places with their locations and a code representing the level of amenity.

Community Services Directory

  • CSV Name: CALGIS.CITY_COMMUNITY_SERVICE.csv
  • Description: It contains all community services with their locations and type os service.

Educational Centers Directory

  • CSV Name: CALGIS.SCHOOL_LOCATION.csv
  • Description: It contains all educational centers categorised by school district, organization type and grades, with location by latitude and longitude.

Definition of the Resource Naming Strategy

This is the URI pattern:
  • Domain: http://www.semanticweb.org/Group32/ontologies/2016/9/
  • Ontology terms: http://www.semanticweb.org/Group32/ontologies/2016/9/#<termName>
  • Individuals: http://www.semanticweb.org/Group32/ontologies/2016/9/<resource_type>/<resource_name>
